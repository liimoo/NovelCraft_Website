# NovelCraft サイト - Claude向けメモ

## プロジェクト概要
- **ノベクラ（NovelCraft）** の公式サイト（静的HTML）
- GitHub Pages でホスティング

## ディレクトリ構成
```
index.html          トップページ（ランディングページ）
assets/
  images/           サイト用画像素材
    hero_bg.png     ヒーロー背景
    classroom.png   教室背景
    school_gate.png 学校の門背景
    chara1〜4.png   キャラクター立ち絵
```

## デプロイ手順
1. ファイルを編集して `git push`
2. GitHub Pages が自動で反映（リポジトリ設定で有効化が必要）
